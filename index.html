shopilo-fr / code-promo-zooplus

Script Python open-source pour la recuperation automatique de codes promo Zooplus depuis shopilo.fr. Renvoie les coupons actifs au format JSON.

Python 3.8+ MIT License requests + beautifulsoup4 shopilo.fr
🔗 Donnees live Zooplus sur shopilo.fr

Installation et utilisation rapide

# Installer les dependances
pip install requests beautifulsoup4

# Cloner le depot
git clone https://github.com/shopilo-fr/code-promo-zooplus
cd code-promo-zooplus

# Executer le script
python fetch.py

Ou installer comme module npm :

npm install code-promo-zooplus

# Utiliser dans Node.js
const { fetchCoupons } = require('code-promo-zooplus');
fetchCoupons().then(data => console.log(data));

Comment fonctionne le script

1Effectue une requete GET vers la page publique https://shopilo.fr/reductions/zooplus.fr avec un User-Agent standard
2Analyse le HTML avec BeautifulSoup et selectionne les elements via des selecteurs CSS (.coupon-item, [data-coupon])
3Extrait pour chaque coupon : le code (data-code), la reduction, la description et la date d'expiration
4Renvoie la liste au format JSON dans stdout, pret pour l'integration dans n'importe quel pipeline, bot ou extension

Questions frequentes, code promo Zooplus

Que renvoie le script fetch.py ?

Un tableau JSON contenant les objets des coupons actifs : le code promo Zooplus, le pourcentage de reduction, la description, la date d'expiration et l'URL source depuis shopilo.fr.

A quelle frequence puis-je executer le script ?

Nous recommandons une execution toutes les 6 heures maximum. Les donnees sur shopilo.fr sont mises a jour quotidiennement, un cron toutes les quelques heures suffit donc.

Fonctionne-t-il aussi comme module npm ?

Oui. npm install code-promo-zooplus installe la version JavaScript qui utilise fetch() natif de Node 18+ et exporte la meme structure JSON.

Ou trouver tous les coupons Zooplus actifs ?

La liste complete est mise a jour quotidiennement sur shopilo.fr/reductions/zooplus.fr, la source de donnees de ce script.

Puis-je combiner plusieurs codes promo Zooplus ?

En general, un seul bon de reduction Zooplus par commande. Les conditions completes sont indiquees sur chaque coupon sur shopilo.fr.

A propos de Zooplus

Zooplus est l'une des boutiques en ligne les plus populaires. Le script recupere automatiquement les codes promo Zooplus disponibles sur shopilo.fr.

Comment utiliser un code Zooplus

1Copiez le code depuis la sortie du script
2Ajoutez les produits souhaites a votre panier Zooplus
3Au moment du paiement, saisissez le code dans le champ "Bon de reduction" ou "Code promo"
4Cliquez sur "Appliquer", la reduction est appliquee automatiquement
